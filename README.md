# pyalea: Python bindings for alpscore::alea

Installation

* prerequisites
  * python 3.8 or later
  * [ALPSCore](https://github.com/ALPSCore/ALPSCore)

* installation
  ```bash
  $ pip install .
  ```

* for development
  ```bash
  $ pip install numpy pybind11 pytest ruff
  $ pip install -ve .
  $ pytest
  ```
